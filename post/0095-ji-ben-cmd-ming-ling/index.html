0095 基本dos命令

发布于 2020-04-02 | 分类于 学习 、 笔记 、 IT | 2分钟 | 327字数 | 浏览量：

这几天命令用的稍微多了点，记录一下学习到的几个常用命令

打开CMD方式

  1. 点击开始，找到windows系统展开，选择命令提示符
  2. win键+R键，在打开的弹框中输入cmd，点击确认
  3. 选中文件夹，按住shift键，点击鼠标右键，选择在此处打开Powershell窗口
  4. 在资源管理器的地址栏上输入cmd或者在地址前面加上cmd，按回车键
  5. 不过推荐使用微软新出的 Windows terminal，兼容cmd、 powershell、Azure、bash，非常好用，具体如何安装可以自行百度。

常用的DOS命令

切换盘符，在命令行输入已存在的盘符，即可切换到对应盘符

D:

查看当前目录下的所有文件

dir

切换目录（change directory）

cd    ::目录名 同一盘符切换
cd /d   ::目录名 跨盘符切换
cd ..     ::返回上一级目录　　

清理屏幕（clear screen）

cls　　

退出终端

exit　

其他命令

ipconfig    ::查看电脑ip
calc    ::打开计算器
mspaint    ::打开画图工具
notepad    ::打开记事本
ping www.baidu.com    ::测试网络是否正常
md    ::文件夹名 创建文件夹（make directory）
cd>   ::文件名 创建文件
del    ::文件名 删除文件
rd    ::文件夹名 删除文件夹（remove directory）
